# 🍽️ Lezzet Diyarı

A modern and responsive restaurant website developed using HTML, CSS, Bootstrap and JavaScript.

The project provides an interactive restaurant experience where users can browse food categories, customize products, manage their shopping cart and apply discount coupons through a clean and user-friendly interface.

---

## ✨ Features

- Responsive modern interface
- Hero landing section
- Interactive food menu
- Product categorization
- Product customization
- Dynamic shopping cart
- Coupon discount system
- Price calculation
- Item removal from cart
- Beautiful animations
- Bootstrap Offcanvas cart
- Bootstrap Modal customization screen
- Font Awesome icons
- Mobile-friendly design

---

## 🛠 Technologies

- HTML5
